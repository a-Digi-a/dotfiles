# Digi's Dotfiles!

These are made for **linux** only

# Installation 

These dotfiles are managed with **YADM** (yet another dotfiles manager):

```shell
sudo pacman -S yadm
yadm clone https://github.com/a-Digi-a/dotfiles.git
yadm status
```

The bootstrap script will install the programs needed for you as well as paru
